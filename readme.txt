== Description ==

Allows you to publishe posts with a delay in your feed. You can set a different delay for each feed type as there are RSS1, RSS2, ATOM and RDF.

As an extra this plugin gives you the possibilities to remove the Wordpress version from your feeds and you can disable/enable each feed type. (This is not working in very old Wordpress version like 2.0.x)

Visit the [Plugin Homepage](http://www.naden.de/blog/feed-delay-wordpress-plugin "Feed delay") for more details.

== Installation ==

1. Unpack the zipfile feeddelay-X.y.zip
1. Upload the folder `feeddelay` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin to fit your needs.
